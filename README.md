# Iconfile

Change default file icon on Windows.

## Screenshot

![screenshot](./assets/screenshot.png)

## Usage

Put svg in `/assets/svg/`.

Convert svg to ico.

`$ python -m converter`

Alter registry.

`$ python -m registry`

Restart system.
